# Projeto — Infinit Code IDE

## Stack padrão
Next.js 14 · TypeScript · Tailwind CSS · Supabase

## Skills disponíveis
- /frontend-design — interfaces únicas e memoráveis
- /ui-ux-pro-max — design system profissional
- /landing-page — landing pages de alta conversão

- Tailwind — nunca CSS inline
- npm run dev → localhost:3000
