# Frontend Design

Antes de qualquer código:
1. **Propósito** — quem usa e por quê
2. **Tom** — minimalista, editorial, brutalist, orgânico, luxury, etc.
3. **Diferencial** — o que torna esta interface inesquecível

## Regras
- Fontes: NUNCA Inter/Roboto/Arial — use Google Fonts com personalidade
- Cores: CSS variables, máximo 2-3 ramps
- Layout: quebre a grade — assimetria, sobreposição
- Motion: 1 animação orquestrada > 10 micro-interações
- Mobile-first sempre
