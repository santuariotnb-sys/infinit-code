# Landing Page — Alta Conversão

## Estrutura obrigatória
1. Hero: headline + subheadline + CTA + prova social
2. Problema: agite a dor em 2-3 linhas
3. Solução: produto como resposta óbvia
4. Features: benefícios (não funcionalidades)
5. Prova Social: depoimentos, logos, números
6. FAQ: elimine objeções
7. CTA Final: urgência + garantia

## Copywriting
- Headline: benefício, não produto
- Use "você" — fale diretamente
- Números específicos > afirmações vagas
- Verbo de ação no CTA
