# UI/UX Pro Max

## Processo
1. Analise: tipo de produto, keywords de estilo, stack
2. Design System: paleta + tipografia + espaçamento consistentes
3. Código: implemente com os tokens definidos

## Estilos
minimal-dark · glassmorphism · neobrutalism · editorial ·
retro-futuristic · soft-pastel · corporate-clean · organic-nature

## Tipografia recomendada
- Display: Playfair Display, Syne, DM Serif, Bebas Neue
- Body: DM Sans, Plus Jakarta Sans, Outfit
- Mono: JetBrains Mono, Space Mono

## Diretrizes UX obrigatórias
- Hierarquia visual em 3 níveis
- Contraste mínimo WCAG AA (4.5:1)
- Touch targets 44x44px mínimo
- Loading states em toda ação assíncrona
