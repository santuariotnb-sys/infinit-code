# Infinit Code

Live Preview, AI Chat e Smart Snippets para VS Code. Kit completo para devs que usam Claude Code.

## Features

### Free
- **Live Preview** — Preview lado a lado para HTML, TSX e JSX. Atualiza a cada tecla.
- **Error Overlay** — Erros de sintaxe aparecem direto no preview.

### Pro (R$67/mês)
- **AI Chat** — Converse com Claude sobre seu código, gere componentes e receba sugestões direto no VS Code.
- **Smart Snippets** — Biblioteca curada de snippets React, Tailwind e CSS prontos para produção.
- **Cloud Sync** — Seus snippets, temas e configs sincronizados em todas as máquinas.
- **Custom Themes** — Crie preview themes com CSS custom.
- **Auto-setup** — Na primeira ativação, instala skills, configura /voice PT-BR, opusplan e preview automático.

## Como usar

1. Abra um arquivo `.html`, `.tsx` ou `.jsx`
2. Pressione `Ctrl+Shift+P` → **Infinit: Live Preview**
3. O preview abre lado a lado com hot reload

## Ativar Pro

1. Assine em [infinitcode.netlify.app](https://infinitcode.netlify.app)
2. Receba sua chave por e-mail (formato: `INFT-XXXX-XXXX-XXXX-XXXX`)
3. `Ctrl+Shift+P` → **Infinit: Activate Pro License**
4. Cole a chave — pronto!

| Comando | Descrição |

## Auto-setup Pro

Na primeira ativação, o Infinit Code configura automaticamente:

- **Skills Pack** — ui-ux-pro-max, frontend-design, supabase-agent, token-optimizer
- **/voice PT-BR** — Fale com o Claude via Meta+K
- **opusplan** — Opus para planejar, Sonnet para codar
- **Preview automático** — Vite + Simple Browser ao abrir o projeto
- **CLAUDE.md + .claudeignore** — Otimizados para seu stack
